WHAT IS AN OPERATING SYSTEM ?

An Operating System is a program, which acts as an interface between a user and the hardware (i.e., all computer resources).
An operating system is an important component of a computer system which controls all other components of a computer system. Major components of a computer system are :

  1. The Hardware
  2. The Software
  3. The Applocation program routines (compiler, linkers, database management systems, utility programms)
  4. The Humanware (users)

An operating system performs mainly following tasks :

  1. Provides user interface
  2. Controls hardware
  3. Assigns Memory
  4. Handles applications
  5. Handles/Manages devices.

FUNCTIONS OF OPERATING SYSTEM

The operating system provides certain services to program and to the users of the programs. Some common services provided by operating system can be listed as follows :

  1. Program Execution
  2. Handling Input/Output Operatons
  3. Manipulation of file system
  4. Error detection and handling
  5. Resources allocation
  6. Accounting
  7. information and Resources Protection

Wild Cards

Two special charcters, '?' (question mark) and '*' (asterisk) are called wildcards in BOSS Linux. They are useful in searching files because they give flexibility in specifying paths and files.

Elements of GNOME Desktop

The GNOME desktop has three distinct areas. From top top bottom, they are :

The Top Menu panel

  2. Right Side contains
    • The Notification area (on top panel-right side)
    • Windows Selection icon (on top panel-right side)

STARTING AND CLOSING PROGRAMS

Most of the programs installed on you computer are available from one convienent location - the Applications menu of the top panel.

To start a program (Say Calculator located on Accessories Group)

  1. Click the Applications menu option on top panel. The Application sub-menu appears.
  2. Point to the group (such as Accessories here) that contains the program you want to start and then click the program name i.e., on Calculator in our case.

To Quit an Open Program

  1. Just click the Close button in the upper- right corner of the program window
  2. Alternatively, you can also click upon Quit option available on the File menu of the application or you can also press the shortcut key Alt+F4.
